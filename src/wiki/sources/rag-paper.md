# Source: Lewis et al. — Retrieval-Augmented Generation

## Source Type

Research paper.

## Summary

Lewis et al. introduce retrieval-augmented generation as a way to connect a generative language model with an external retrieval system.

The model can use retrieved passages when generating answers, which helps it work with information beyond what is stored in its parameters.

## Key Ideas

- Language models store knowledge in parameters, but this can be limited.
- External retrieval can provide access to explicit knowledge.
- RAG combines retrieval with generation.
- Retrieved passages can improve specificity and factuality.
- RAG is especially useful for question answering and knowledge-intensive tasks.
